### Install

Add jitpack to your root `build.gradle`
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```
	
Add the dependency
```
	dependencies {
	        compile 'com.github.metagalactic:ScalableImageView:0.0.1'
	}
```
